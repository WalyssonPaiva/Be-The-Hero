<h2>:information_source: How To Use </h2>

From your terminal:

<h5> Install API</h5>

``` 
Clone this repository
$ git clone https://github.com/WalyssonPaiva/Be-The-Hero.git

Go into the repository
$ cd Be-The-Hero/backend

Install dependencies
$ npm i

Run Migrates
$ npx knex migrate:latest 

Run the API
$ npm start

```
